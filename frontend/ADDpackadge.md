План изучения стека (CRM-Farma)

    1-Prisma (pnpm prisma init) — ORM для работы с базой данных. Учу: схемы, миграции и запросы.
    2-Tailwind CSS — Стилизация. Учу: утилитарные классы, адаптив и v4 конфиг.
    3-React Router (react-router-dom) — Навигация. Учу: маршруты, вложенные роуты (Outlet) и хуки useNavigate/useParams.
    4-Zustand — Глобальный стейт. Учу: создание стора, экшены и работу без провайдеров.
    5-TanStack Query — Работа с API. Учу: useQuery для получения данных и useMutation для отправки.
